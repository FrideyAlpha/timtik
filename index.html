SiLAP v.2

SiLAP adalah aplikasi pendukung bidang kesekretariatan dari Bagian SDM - Sekretariat DJPb.


Aplikasi ini memiliki fungsi untuk pengumpulan data absensi para pegawai Ditjen Perbendaharaan diseluruh satker, seluruh Indonesia.

Fungsi umum aplikasi ini adalah untuk menjembatani proses pertukaran data antara perangkat absensi seperti Handkey dan Palm Vein Reader (PVR) yang digunakan satker kita sebagai device pencatat kehadiran pegawai, serta pembuatan laporan kepegawaian dalam berbagai format.

Aplikasi ini juga terintegrasi dengan aplikasi lain seperti SIPKK, untuk keperluan pencegahan duplikasi pembayaran tagihan pegawai seperti Lembur dan Perjalanan Dinas, Uang Makan dan RDK, serta verifikasi jumlah jam kehadiran pada kegiatan RDK.

User pengguna aplikasi ini adalah operator pada masing-masing satker, operator tingkat wilayah, dan Admin pada Kantor Pusat

Aplikasi ini pertama dikembangan pada tahun 2009 oleh Tim Inovasi TIK - Sekretariat DJPb.

SIPAT v.4

SIPAT adalah aplikasi pendukung bidang kesekretariatan dari Bagian Umum - Sekretariat DJPb.


Aplikasi ini memiliki fungsi untuk mendukung sisi manajerial dari manajemen aset, membantu para pengelola aset mengambil kebijakan seputar pengelolaan aset.

Fungsi umum aplikasi ini adalah untuk menampilkan informasi-informasi terkait pengelolaan aset, seperti lokasi aset strategis berdasarkan lokasi koordinat, demo grafis BMN, foto gallery untuk seluruh aset, dan master data untuk aset sesuai data pada Aplikasi SIMAN - DJKN.

Aplikasi ini juga terintegrasi dengan aplikasi lain seperti SAKURA, untuk keperluan perencanaan dan kebutuhan anggaran satker DJPb di seluruh Indonesia.

User pengguna SIPAT adalah seluruh pegawai, masing-masing operator pada satker, Admin pada Kantor Pusat dan para pejabat pengelola aset.

Aplikasi ini pertama dikembangan pada tahun 2010 oleh Tim Inovasi TIK - Sekretariat DJPb.

SIPKK v.3

SIPKK adalah aplikasi pendukung kinerja pengelolaan keuangan dari Bagian Keuangan - Sekretariat DJPb.


Aplikasi ini memiliki fungsi untuk memanage siklus tagihan, dari kegiatan dilaksanakan hingga tagihan dibayarkan.

Fungsi umum aplikasi ini adalah untuk mengawasi pengelolaan keuangan di Kantor Pusat, dengan modul-modul yang sudah dikembangkan antara lain Modul Laporan Realisasi, Tagihan, Honor Tim, Perjalanan Dinas, Mutasi Pegawai, Capaian Kinerja, Konsumsi Rapat, dan Pra SPP. Dengan adanya aplikasi ini tidak dimungkinkan lagi ada gagal bayar karena dokumen yang tercecer karena aplikasi menggunakan fitur e-filing, juga fitur tersendiri untuk membatasi adanya dobel pembayaran atas kegiatan yang berbeda untuk satu penerima yang tidak diperbolehkan, efisiensi pembayaran perjalanan dinas mutasi dengan otomatisasi nilai perjalanan berdasarkan rute, serta optimalisasi kinerja dengan menggunakan fitur capaian kinerja yang menghubungkan antara kinerja user aplikasi capaian kinerjanya.

Aplikasi ini juga terintegrasi dengan aplikasi lain seperti SiLAP, untuk keperluan verifikasi potensi fraud atas pembayaran tagihan kegiatan seperti RDK dan Perjalanan Dinas, Lembur, Uang Makan dengan absensi pegawai.

User pengguna SIPKK adalah PIC Kegiatan, PIC Konsumsi Rapat, PIC Keuangan, Admin pada Kantor Pusat dan para pejabat pengelola keuangan.

Aplikasi ini pertama dikembangan pada tahun 2013 oleh Tim Inovasi TIK - Sekretariat DJPb.

iPRO v.1

iPRO adalah aplikasi demografis dari Bagian OTL - Sekretariat DJPb.


Aplikasi ini memiliki fungsi untuk menampilkan data demografi dari berbagai macam aspek seperti Realisasi, SDM, Aset, Beban Kinerja, dll.

Fungsi umum aplikasi ini adalah untuk mengoptimalkan tampilan data dari seluruh aplikasi pendukung kesekretariatan lainnya seperti SIPAT, SAKURA, SIPKK, SiLAP, DAMS-DJPB, dll. Data dari aplikasi-aplikasi tersebut terintegrasi satu sama lain dan disajikan dalam bentuk demografis pada aplikasi iPRO.

Sasaran pengguna pada aplikasi ini adalah level manajer pada Ditjen Perbendaharaan.

User pengguna iPRO hanya ada pada Bagian OTL - Kantor Pusat. Sementara data diprovide oleh seluruh aplikasi kesekretariatan lainnya.

Aplikasi ini pertama dikembangan pada tahun 2013 oleh Tim Inovasi TIK - Sekretariat DJPb dengan nama iProud, kemudian diubah nama menjadi iPRO pada tahun 2019.

DAMS-DJPb v.1

DAMS-DJPb adalah aplikasi demografis dari Bagian OTL - Sekretariat DJPb.


Aplikasi ini memiliki fungsi untuk mengelola kinerja harian pegawai Ditjen Perbendaharaan.

Fungsi umum aplikasi ini adalah mengawasi kinerja harian pegawai, dalam format Matriks Tindak Lanjut (MTL) pada Daily Activity Monitoring System (DAMS). Tugas dari seorang atasan kepada bawahannya dapat di-assign melalui aplikasi ini, dengan parameter lingkup waktu pengerjaan, sasaran kerja, tim project, anggaran, progress penyelesaian, dan parameter penyelesaian suatu pekerjaan. Seluruh progress pada MTL di aplikasi ini (Perekaman MTL, Update Progress, Diskusi, dll) didukung notifikasi melalui pesan daring Whatsapp ke nomor masing-masing pengguna, dari level pejabat Eselon I hingga level Pelaksana yang terlibat dalam satu MTL.

User pengguna iPRO adalah seluruh pegawai dan level manajer pada Ditjen Perbendaharaan.

Aplikasi ini pertama dikembangan pada tahun 2019 oleh Tim Inovasi TIK - Sekretariat DJPb

DJPb Meeting

DJPB Meeting adalah aplikasi Pertemuan Tatap Muka secara Daring yang dikembanglan oleh Tim Inovasi TIK - Sekretariat DJPb.


Aplikasi ini memiliki fungsi memfasilitasi pertemuan tatap muka secara daring, yang saat ini tidak dapat dihindarkan dalam perkembangan era New Normal.

Fungsi umum aplikasi ini adalah melakukan panggilan video conference dengan jumlah Ruang Rapat dan Peserta tidak dibatasi, dan dengan waktu tak terbatas. Pertemuan direkam dalam 2 format media yaitu Video dan Audio Only. Aplikasi dapat digunakan tanpa software tambahan apapun pada perangkat Desktop/Mobile pengguna, cukup dengan browser, kamera, dan mikropon saja. Aplikasi ini menggunakan server Sekretariat DJPB, sehingga memiliki jaminan keamanan maksimal pada penggunanya.

User pengguna DJPb Meeting adalah seluruh pegawai pada Ditjen Perbendaharaan.

Aplikasi ini pertama dikembangan pada tahun 2020 oleh Tim Inovasi TIK - Sekretariat DJPb

SAKURA

SAKURA adalah aplikasi perencanaan anggaran internal Ditjen Perbendaharaan oleh Bagian Keuangan - Sekretariat DJPb.


Aplikasi ini memiliki fungsi manajerial sebagai pengolah informasi pendukung pengambilan keputusan terkait perencanaan anggaran Ditjen Perbendaharaan.

Fungsi umum aplikasi ini adalah menampilkan data dukung yang diperlukan dalam penyusunan anggaran seperti Jumlah Satker, Data Pegawai, Realisasi Anggaran, Data BMN, Perencanaan Kebutuhan BMN, Rencana Penghapusan BMN, dsb. Data tersebut diprovide oleh aplikasi kesekretariatan lainnya seperti SiLAP, SIPKK dan SIPAT.

User pengguna SAKURA adalah seluruh operator SAKURA pada masing-masing satker lingkup Ditjen Perbendaharaan.

Aplikasi ini pertama dikembangan pada tahun 2015 oleh Tim Inovasi TIK - Sekretariat DJPb

Whatsapp Services

Whatsapp Services adalah fitur integrasi Media Perpesanan Daring Whatsapp dengan Aplikasi Kesekretariatan yang dikembangkan oleh Tim Inovasi TIK - Sekretariat DJPb.


Fitur ini memiliki fungsi mengintegrasikan event-event yang dibutuhkan pada Aplikasi Kesekretariatan yang dapat memanfaatkan pesan daring Whatsapp kepada penggunanya.

Fungsi umum aplikasi ini mengirimkan notifikasi dari aplikasi, seperti dokumen masuk, tagihan yang telah terbayar, serta penugasan MTL pada pegawai. Mengingat pada umumnya aplikasi pendukung tusi tidak terpantau notifikasinya jika aplikasinya sendiri tidak dibuka/diakses. Fitur ini menjembatani permasalahan akses dari pengguna ke aplikasi, sehingga pengguna tidak kehilangan informasi pada aplikasi yang ditanganinya.

User pengguna fitur ini adalah seluruh pegawai satker lingkup Ditjen Perbendaharaan.

Aplikasi ini pertama dikembangan pada tahun 2019 oleh Tim Inovasi TIK - Sekretariat DJPb

DJPb Cloud Storages

DJPb Cloud Storages adalah fitur penyimpanan arsip dan file berbasis Cloud dari Tim Inovasi TIK - Sekretariat DJPb.


Fitur ini memberikan space sebesar 5GB untuk seluruh penggunanya, penyimpanan arsip berbasis Cloud yang dapat diakses dari mana saja untuk seluruh pegawai Ditjen Perbendaharaan.

Fungsi umum aplikasi ini adalah menjadi tempat penyimpanan digital yang dapat diakses dimana saja, memudahkan konsep Flexible Working Space yang akan diterapkan oleh pegawai Ditjen Perbendaharaan di masa New Normal ini.
